Deploy via GitHub Pages and use:

## Features

- **Dashboard** — Income, expense, balance, budget progress, charts
- **Transactions** — Add, edit, delete with filters
- **Income & Expense** — Separate categories for each type
- **Monthly Budget** — Set and track spending limits
- **Reports** — Pie chart, bar chart, savings rate
- **Export** — Download transactions as CSV
- **Dark / Light Theme** — Toggle with saved preference
- **Responsive** — Works on desktop, tablet, and mobile

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5
- Chart.js
- localStorage (client-side data persistence)

## Getting Started

### Option 1: Open directly

Double-click `index.html` in your browser.

### Option 2: Local server (recommended)

```bash
cd expense-tracker
python -m http.server 8080
```

Open `http://localhost:8080`
